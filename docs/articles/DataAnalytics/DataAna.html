Data Analytics

In addition to loading and retrieving data, RIVALYTICS allows you to perform basic data analysis.

Our data analytics tools are available as worksheet functions. This means, that they can be accessed like any other Excel function such as SUM, IF or UNIQUE (see Figure 1).

Figure 1: KMEANS Worksheet functions

They help you perform various data analytics tasks like clustering or classification. In addition to algorithms like K-Means Clustering or the Naive Bayes Classifier, the library also contains helper functions like TERMFREQUENCIES or DOCUMENTTERMMATRIX.
